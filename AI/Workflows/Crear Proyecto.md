# Workflow: Crear Proyecto

## Pasos

### 1. Analizar
- Leer el objetivo del proyecto
- Identificar requisitos y restricciones
- Documentar en AI/Memory/proyectos.md

### 2. Planificar
- Usar AI/Prompts/planner.md para generar plan
- Definir estructura de carpetas
- Elegir stack tecnológico

### 3. Implementar
- Usar persona Developer
- Seguir AI/Rules/coding.md
- Hacer commits atómicos

### 4. Documentar
- Actualizar README.md
- Añadir docs necesarios en Documentation/

### 5. Finalizar
- Verificar que todo funciona
- Actualizar AI/Memory/proyectos.md
- Commit final con mensaje descriptivo
