# Workflow: Solución de Bugs

## Pasos

### 1. Identificar
- Describir el bug
- Reproducirlo

### 2. Debuggear
- Usar AI/Prompts/debugger.md
- Identificar causa raíz

### 3. Corregir
- Implementar fix
- Verificar que no se rompe nada más

### 4. Documentar
- Actualizar changelog si aplica
- Commit con tipo "fix:"
