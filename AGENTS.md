# OpenCode Ultimate — Instrucciones para la IA

## Proyecto
OpenCode Ultimate: Entorno profesional de desarrollo con IA, memoria persistente y automatización.

## Memoria persistente
- `AI/Memory/perfil.md` — Perfil del desarrollador
- `AI/Memory/preferencias.md` — Preferencias técnicas
- `AI/Memory/proyectos.md` — Lista de proyectos activos

## Personas disponibles
- `AI/Personas/Software Architect.md` — Para diseño y arquitectura

## Reglas
- `AI/Rules/coding.md` — Reglas de codificación

## Prompts
- `AI/Prompts/planner.md` — Prompt para planificar tareas
- `AI/Prompts/debugger.md` — Prompt para depuración

## Workflows
- `AI/Workflows/Crear Proyecto.md` — Workflow para nuevos proyectos

## Comportamiento esperado
1. Antes de cualquier tarea, lee los archivos de memoria y reglas.
2. Usa la persona adecuada según la tarea.
3. Sigue los workflows cuando apliquen.
4. Al finalizar, pregunta si debe actualizar la memoria.
5. Usa español para comunicarte con el usuario.
